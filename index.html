Waarom dit werkt

Je lichaam leert efficiënter bewegen. Dat levert direct output op: meer snelheid, meer power, meer robuustheid. Geen ruis — alleen fundamentals + elite uitvoering.

Wat zit erin

  • ✓
    Intake + Testdag Kennismaken + testen (o.a. sprint, HR, kracht). Start met duidelijke baseline.
  • ✓
    11 dagen training (90–120 min) Veld • kracht • body movements — strak geprogrammeerd.
  • ✓
    Exit + evaluatie We meten progressie en zetten je richting seizoenstart.

Unieke invalshoek

“Ontdekken hoe je lichaam werkt” → optimaal gebruiken → sneller, explosiever en krachtiger worden. Functioneel bewegen staat centraal.
